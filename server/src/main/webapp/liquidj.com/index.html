LiquidJ

Beyond ajax -- shared objects for javascript.

LiquidJ provides transparent replication of object state between browser and server.

Modify an object in the browser and it's sent to the server automatically. Modify an object in the server and it's transparently sent to the browser.

You can use LiquidJ to:
  • Easily persist javascript objects to the server. You just define the data types you care about, and LiquidJ does change detection and ajax saving/loading automatically.
  • Drop in persistent UI widgets. LiquidJ enhanced Javascript UI widgets save themselves automatically, and update live via Comet. LiquidJ enhancements to jQuery.UI widgets are included, and it's easy to data enhance your favorite widget too.
  • Create richer javascript apps by separating data from markup. LiquidJ helps enable a programming style that moves dynamic view logic out of the server and into the browser.
